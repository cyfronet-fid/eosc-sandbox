For Pilot EOSC Nodes and other providers

If you are representing a potential new EOSC Node or a service provider, you can use the EOSC Innovation Sandbox to integrate your Node or service with EOSC and test and validate your work before moving into production.
A team of technical experts is available to support you in this process. Please, contact us shortly describing your case and the type of integration you would like to achieve (e.g. setting up a new EOSC node for the structural biology community, publishing data in EOSC, offering cloud resources to EOSC, etc.). Contact us

Currently, the following federating capabilities are offered for integration:

AAI
• Allows users to log in with their own institute credentials (eduGAIN) to access EOSC resources and enables the single sign-on over nodes of the EOSC Federation.
Accounting
• Provide information about usage of services and research products (e.g. dataset) in EOSC (Core and Exchange).
Execution Framework
• Enables the onboarding of deployable applications in EOSC and manage their execution in the underlying infrastructures.
Helpdesk
• Federation of integrated helpdesks to request support for all EOSC services.
Integration Suite
• Set of software libraries (adapters) that facilitate the process to integrate services in EOSC.
Messaging
• Enable efficient communication between EOSC services.
Monitoring
• Collect and publish information about the availability and reliability of EOSC services.
PID
• Assign and resolve PIDs to providers and resources onboarded in EOSC.
Resource Catalogue
• Federated EOSC Resource Catalogue making the Node exchange resources discoverable.
• Search engine over the whole EOSC Federation.
• Interfaces to allow EOSC Nodes to publish (push) and retrieve (pull) resources to/from the federated catalogue
Order Management
A framework for:
• Providers: to define offers for their services
• End-users: common interface to request access to the EOSC services
User space
• Dynamic customisable web frontend, where the node user logs-in, offering capabilities to discover and access the Node resources.
