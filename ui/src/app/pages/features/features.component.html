Main features

Federation Capabilities

EOSC EU Node: the Sandbox offers for integration the same federating capabilities delivered by the EOSC EU Node (e.g. the AAI Federation, the Federated Resource Catalogue, the Helpdesk, the Application Workflow Management, etc.) with compatible interfaces for testing.

Additional Federating Capabilities developed by the project.

Technical support

Assist providers willing to integrate with EOSC with dedicated technical support and offer tools to automate the integrations (e.g. easy-to-use connectors in the form of software libraries).

Showcase new EOSC Core capabilities

Research communities and scientists can test and assess the new EOSC capabilities that EOSC Beyond delivers.
