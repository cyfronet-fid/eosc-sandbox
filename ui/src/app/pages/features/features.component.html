Main features

Federation Capabilities

The Sandbox offers for integration existing and new federating capabilities of the EOSC Federation with compatible interfaces for testing: AAI, the Federated Resource Catalogue, the Helpdesk, the Application Workflow Management, etc.

Technical support

Assist candidate nodes and providers willing to integrate with EOSC with dedicated technical support and offer tools to automate the integrations (e.g. easy-to-use connectors in the form of software libraries).

Showcase new EOSC Core capabilities

Research communities and scientists can test and assess the advanced EOSC capabilities that EOSC Beyond delivers.
